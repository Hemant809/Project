1. Enter Inputs

You provide values such as frequency and amplitude for both the message and carrier signals. You also select the waveform type (sine or cosine) and choose a modulation technique (AM, FM, PM, SSB, DSB-SC, ASK, FSK, QPSK, or BPSK).

2. Signal Generation

The system generates both the message signal and the carrier signal using your input parameters. It then plots visual waveforms for each signal.

3. Modulation

Based on your selected modulation type:

  • AM (Amplitude Modulation): Varies carrier amplitude with the message signal.
  • FM (Frequency Modulation): Varies carrier frequency with the message signal.
  • PM (Phase Modulation): Varies carrier phase with the message signal.
  • SSB (Single Side Band): Transmits only one sideband (upper or lower) of the AM signal.
  • DSB-SC (Double Side Band – Suppressed Carrier): Sends both sidebands but removes the carrier.
  • ASK (Amplitude Shift keying): Changes carrier amplitude between discrete levels based on digital input.
  • FSK (Frequency Shift keying): Changes carrier frequency between discrete levels based on digital input.
  • QPSK (Quadrature Phase Shift Keying): Sends data by changing the carrier phase in four distinct steps.
  • BPSK (Binary Phase Shift keying): Sends data by changing the carrier phase between two values (0° and 180°).

4. Visualization

The resulting modulated signal is displayed along with the original input signals as plots. This helps in understanding how modulation affects signal shape.
